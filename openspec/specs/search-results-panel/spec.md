## ADDED Requirements

### Requirement: Dynamic search results panel
A panel SHALL appear directly beneath the search bar to display a list of matching facilities.

#### Scenario: Search yields results
- **WHEN** the user types a query matching one or more facilities
- **THEN** the search results panel becomes visible under the search bar displaying the matching items

#### Scenario: Search box is empty
- **WHEN** the search box is cleared
- **THEN** the search results panel is hidden
