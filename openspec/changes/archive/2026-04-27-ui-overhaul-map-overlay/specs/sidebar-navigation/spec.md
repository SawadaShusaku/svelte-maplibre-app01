## ADDED Requirements

### Requirement: Settings sidebar navigation
The sidebar SHALL be used exclusively for application settings and navigation, accessible via a hamburger icon.

#### Scenario: User opens sidebar
- **WHEN** the user clicks the hamburger menu icon
- **THEN** the sidebar slides in showing navigation and settings options

#### Scenario: Facility list removed from sidebar
- **WHEN** the sidebar is open
- **THEN** it does NOT display the list of recycling facilities or category filters
