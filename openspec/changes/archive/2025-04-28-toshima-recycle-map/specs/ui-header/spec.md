## ADDED Requirements

### Requirement: 固定ヘッダーに見出しを表示
アプリ画面の最上部に固定ヘッダーを設け、見出しを表示しなければならない（SHALL）。

#### Scenario: 初期表示
- WHEN ユーザーがページを開く
- THEN ヘッダーが画面上部に表示され、タイトルとして「東京リサイクルマップ：豊島区版」が見える

#### Scenario: 将来の区切替
- WHEN 他区に切り替える
- THEN 見出しは「東京リサイクルマップ：{区名}版」に更新される

### Requirement: 見出しのタイポグラフィ
見出しには日本語フォント「Dela Gothic One」を用いなければならない（SHALL）。本文フォントは従来のSans系とし、見出し用途に限定する。

#### Scenario: フォント配信
- WHEN フォントを読み込む
- THEN Google Fonts などの配信を用い、`font-display: swap` と `preconnect` を設定してパフォーマンスと表示安定性を確保する

### Requirement: レイアウト/アクセシビリティ
ヘッダーは地図コンテンツと重なる場合でも可読性が維持され、キーボード/スクリーンリーダーで見出しとして認識されなければならない（SHALL）。

#### Scenario: モバイルでの重なり
- WHEN モバイル端末で表示する
- THEN ヘッダーが地図のUIコントロール（ズーム/現在地/全画面）と干渉しない
