## ADDED Requirements

### Requirement: カテゴリフィルタUI
画面上にカテゴリ選択UIを表示しなければならない（SHALL）。ユーザーは1つ以上のカテゴリを選択でき、地図上のマーカーが絞り込まれる。

#### Scenario: 全カテゴリ選択（初期状態）
- **WHEN** ページを開いた直後
- **THEN** 全カテゴリが選択済み状態となり、全拠点が表示される

#### Scenario: 単一カテゴリ絞り込み
- **WHEN** ユーザーが「乾電池」のみを選択する
- **THEN** 乾電池を収集する拠点のマーカーのみが表示される

#### Scenario: 複数カテゴリ絞り込み
- **WHEN** ユーザーが「乾電池」と「小型家電」を選択する
- **THEN** 乾電池または小型家電を収集する拠点のマーカーが表示される（OR条件）

#### Scenario: 全カテゴリ解除
- **WHEN** ユーザーが選択中の全カテゴリを解除する
- **THEN** マーカーは1件も表示されない
